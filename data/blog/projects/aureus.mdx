Aureus is a design system / SDK built in Flutter that was meant to help people or non-profits quickly build user safety critical consumer apps. We also used it internally to rebuild our tech stack into one language for cohension and logistics.

[Find Aureus on Github here.](https://github.com/Astra-Labs/Aureus)
